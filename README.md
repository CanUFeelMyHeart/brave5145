# Инструкция по работе с markdown

## Выделение текста

Чтобы выделить текст курсивом нужно бахнуть звезду (*) Пример: *прорыв*

выделить жиром: **Жир**
 или бабахнуть _вот так_
 а можно и __так__

## Списки
чтобы текст ебался прилично

Чтобы бахнуть списки, надо вставить звезды
 * вот

 * пример

Чтобы с цифрами
1. вот
2. пожалуйста
